NAME
----
ml-lex - a scanner generator for *Standard ML*.

SYNOPSIS
--------
*ml-lex* 'file'

DESCRIPTION
-----------

*ML-Lex* is a scanner generator for *Standard ML*.

This software is deprecated; we recommend using the *ml-ulex*(1) tool
in *ml-lex*-compatibility mode instead.
